

### Cargo
----
```bash
cargo init --bin | --lib
```
-> create new project

```bash
cargo new <name> --lib
```
-> new a lib

```
cargo run
```
-> run this project

```bash
cargo build --release
```
-> build release

### Rust fmt
----
```bash
cargo fmt
```
-> format
```bash
rustfmt <file_path>
```
-> format
```bash
rustfmt src/*.rs
```
-> format file is rust
